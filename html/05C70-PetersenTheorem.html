Petersen theorem


Every finite , 3-regular (http://planetmath.org/Valency), 2-edge connected graph has a complete matching.

Proof.

Using the notations from the Tutte theorem, we have to prove that for all X⊆V⁢(G) the inequality cp⁢(G-X)≤|X| holds. There are at least 3 edges running between X and an odd component of G-X: there cannot be one edge, since G is 2-edge connected, and there also cannot be two edges, because three edges start from all vertices of an odd component, so the number of edges leaving an odd component is odd. Let t be the number of all edges between X and the odd components of G-X. Now we have t≥3⁢cp⁢(G-X). But G is 3-regular, thus t≤3⁢|X|. This gives cp⁢(G-X)≤|X|. ∎
